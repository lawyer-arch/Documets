Часть 1. Работа с БД/SQL

Задание 1.1: Написать SQL запрос, который выведет код и наименование всех опций, стоимость которых более 900, в порядке возрастания цены. Решение: SELECT o.code, o.name, op.price FROM option o JOIN option_price op ON o.id = op.option_id WHERE op.price > 900 ORDER BY op.price ASC; Пояснение: - соединяем таблицы option и option_price по ID опции; - фильтруем записи с ценой более 900; - сортируем результат по возрастанию цены. Задание 1.2: Написать SQL запрос, который выведет количество клиентов, наименование которых начинается на ООО Решение: SELECT COUNT(*) AS client_count FROM client WHERE name LIKE ‘ООО%’; Пояснение: - используем функцию COUNT для подсчета записей; - применяем оператор LIKE с шаблоном ‘ООО%’ для поиска названий. Задание 1.3: Написать SQL запрос, который выведет наименование и ИНН клиентов, у которых подключена опция «Маркетплейс» Решение: SELECT c.name, c.inn FROM client c JOIN client_option co ON c.id = co.client_id JOIN option o ON co.option_id = o.id WHERE o.name = ‘Маркетплейс’; Пояснение: - cоединяем таблицы client, client_option и option; - фильтруем записи по названию опции. Задание 1.4: Написать SQL запрос, который выведет наименование и ИНН клиентов в первой колонке с названием «Клиент (ИНН)» в формате: ООО «Росинка» (8394119283), во второй колонке с названием «Сумма за обслуживание» сумму, которую клиент платит за обслуживание тарифа (price из таблицы tariff), а также за все подключенные опции (price из таблицы option_price). Решение: SELECT CONCAT(c.name, ‘(‘, c.inn, ‘)‘) AS ‘‘Клиент (ИНН)’’, (t.price + COALESCE(SUM(op.price), 0)) AS ‘‘Сумма за обслуживание’’ FROM client c JOIN tariff t ON c.tariff_id = t.id LEFT JOIN client_option co ON c.id = co.client_id LEFT JOIN option_price op ON co.option_id = op.option_id GROUP BY c.id, t.price; Пояснение: - используем CONCAT для формирования строки с ИНН - суммируем цену тарифа и всех подключенных опций - COALESCE используется для обработки случаев без опций; Задание 1.5: Написать SQL запрос на добавление записи в таблицу client: Наименование: ООО «Крылатые качели» ИНН: 8823710320 Тариф: Стартовый Решение: INSERT INTO client (name, inn, tariff_id) VALUES ( ‘ООО «Крылатые качели»‘, 8823710320, (SELECT id FROM tariff WHERE name = ‘Стартовый‘) ); Пояснение: - вставляем данные в таблицу client; - используем подзапрос для получения ID тарифа; - указываем все обязательные поля для вставки.
